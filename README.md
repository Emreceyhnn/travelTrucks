# TravelTrucks

A modern web application for browsing and booking camper rentals, built with React, Redux Toolkit, and TypeScript.

## Features

- **Home Page** — Hero banner with CTA leading to the catalog
- **Catalog** — Browse all available campers with backend filtering (location, vehicle type, equipment)
- **Favorites** — Save campers to favorites (persisted across page reloads)
- **Pagination** — Load more campers on demand
- **Camper Details** — Photo gallery, Features/Reviews tabs, booking form
- **Star Ratings** — 5-star rating display for user reviews
- **Booking Form** — Reserve a campervan with form validation and success notification

## Tech Stack

- **Vite** + **React** + **TypeScript**
- **Redux Toolkit** for state management
- **React Router** for routing
- **Axios** for API requests
- **CSS Modules** for scoped styling
- **react-hot-toast** for notifications

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
